# CaveManIDE

CaveManIDE is a **minimalistic Python IDE** built with `customtkinter` and `tkinter`. It is designed to provide a lightweight interface for writing, saving, opening, and running Python scripts quickly and efficiently.  

---

## Features

- **Minimalist Interface**: Simple and clean design to focus on coding.  
- **Save & Open Files**: Easily save your Python scripts or open existing ones.  
- **Run Code**: Execute Python scripts directly from the IDE.  
- **Semi-Transparent Window**: Modern look with adjustable transparency.  
- **Output Display**: View your script output in a dedicated read-only console.  

---

## Installation

1. Make sure you have **Python 3.x** installed.  
2. Go to main and install CaveManIDE.py
3. Install neccesary modules
```bash
pip install customtkinter
```
4. Run It
```bash
python CaveManIDE.py
```

## Usage

Save: Click the "Save" button to save your current script.

Open: Click the "Open" button to open an existing Python file.

Run: Click the "Run" button to execute your code. Output will appear in the gray console at the bottom.
